서버 엔지니어 모집

18.02.21 ~ 채용시

왓 스튜디오의 서버아키텍처팀은

이런 일을 합니다

  • 새로운 서버 프레임워크를 개발합니다
  • 목표하는 서버 프레임워크는 다음과 같은 목표를 가집니다
    • 도메인별 데이터를 분산 환경에서 효율적으로 다룹니다
    • 트래픽에 따라 서버의 수평적 확장이 가능해야 합니다
    • 데이터를 도메인별로 일관성이 깨지지 않게 합니다
  • 개발하는 프레임워크는 다음과 같은 특징을 가지고 있습니다
    • .NET Core 기반으로 제작되고 있습니다
    • 액터 모델을 기반으로 제작되고 있습니다

이렇게 일을 합니다

  • Git 친화적 개발 환경
    • 버전 관리 도구로 Git을 사용하며, 브랜치는 Git Flow 방식으로 관리합니다.
    • CI/CD는 모두 Git과 연계됩니다.
    • Git을 올바른 방식으로 사용하려고 노력합니다.
    • Git에 익숙하지 않은 사람이더라도 올바른 방향으로 사용할 수 있도록 훈련할 수 있는 환경이 제공됩니다.
  • 코드 리뷰 시스템
    • 팀에서 작업한 모든 코드는 Merge Request(Pull Request) 방식을 통해 메인 저장소에 반영됩니다.
    • 메인 저장소에 반영되기 위해서는 다른 동료의 코드 리뷰가 필수적입니다.
    • 코드 리뷰 과정에서는 작게는 변수명부터 크게는 설계 구조까지 더 나은 코드를 만들기 위한 피드백을 주기 위해 노력합니다.
    • 코드 리뷰는 GitLab의 코드 리뷰 시스템을 사용합니다.
  • 테스트 친화적인 환경
    • CI와 연계된 자동화 테스트 시스템이 구축되어 있고, 계속해서 발전시키기 위해 노력합니다.
    • 새로운 기능을 만들 때, 유닛 테스트를 만드는 것을 권장하는 문화가 자리 잡고 있습니다.
  • 오픈소스 친화적인 환경
    • 오픈소스의 기술들을 적극적으로 수용하고 있습니다.
    • 하지만 오픈소스를 일방적으로 수용하는 것을 경계하며, 합리적인 판단 과정을 거치려고 노력합니다.
    • 기회가 될 때마다 오픈소스 문화에 기여할 수 있도록 노력합니다.

이런 환경에서 일합니다

  • 조직 내 커뮤니케이션을 위해 Slack을 사용합니다.
  • 지식 공유와 문서화를 위해 Confluence를 사용합니다.
  • 업무와 일정 관리를 위해 Jira를 사용합니다.
  • 코드 리뷰와 CI/CD 연계를 위해 설치형 GitLab EE를 사용합니다.
  • 서비스 인프라로 Amazon Web Services (AWS) 클라우드 서비스를 사용합니다.
  • 유연 근무제를 시행하고 있습니다. 7시~10시 사이 출근이 가능합니다.
  • 다른 인사제도와 복리후생은 넥슨코리아의 정책을 따릅니다.

이런 동료와 함께 일하고 싶습니다

  • 좋은 코드를 짜려는 마음가짐을 갖고 노력을 하는 분
  • 수동적이지 않고 스스로 할 일을 정할 수 있는 분
  • 직군과 관계없이 커뮤니케이션이 가능하신 분
  • 불편한 상황을 수용하지 않고 개선하려는 의지를 갖고 노력하는 분
  • 합리적인 판단 아래 새로운 기술을 탐색하고 도입하려고 노력하는 분

이런 분을 찾고 있습니다

  • 새로운 서버 기술 도입에 적극적이신 분
  • C#, .NET Core에 익숙하거나 관심이 있는 분
  • 네트워크에 대한 기본적인 이해도가 있는 분

다음과 같은 조건은 필수는 아니지만 갖추신 분들은 우대합니다

  • 백엔드 엔지니어나 서버 엔지니어 1~2년 이상의 경력을 가지고 계신 분
  • 오픈소스 사용에 익숙하거나 관심이 있는 분
  • 테스트와 자동화를 선호하시는 분
  • Linux 환경에서의 시스템 운용 경험이 있는 분
  • CI/CD 구축, 운용 경험이 있는 분
  • Container 기반 개발, 운용 경험이 있는 분
  • Python 개발 경험이 있는 분
  • 퍼블릭 클라우드에 익숙하신 분
  • 마이크로서비스(Microservices) 개발 경험이 있는 분
  • NoSQL에 대한 이해도가 높으신 분 -〈야생의 땅: 듀랑고〉를 즐기시는 분

기타사항

  • 국가보훈대상자는 관련법규에 의거, 우대합니다
  • 서류 심사 통과자에 한해 개별적으로 일정을 알려 드립니다
